## Using
For more details, check the [NUR README](https://github.com/nix-community/nur#readme)
### A package (see [the package list](#packages-list))
```shell_session
$ nix-shell -p <attribute path>
```
Example : 
```shell_session
$ nix-shell -p nur.repos.shamilton.sync-database
```

### A NixOS module
In `/etc/nixos/configuration.nix` :
```nix
let
  # Please edit this and replace with a fixed revision
  # and sha256
  nur-no-pkgs = import (builtins.fetchTarball "https://github.com/nix-community/NUR/archive/master.tar.gz") {};
in
{
  imports = [
	nur-no-pkgs.repos.shamilton.modules.numworks
  ];
}
```

### A home-manager module
In `~/.config/nixpkgs/home.nix` :
```nix
let
  # Please edit this and replace with a fixed revision
  # and sha256
  nur-no-pkgs = import (builtins.fetchTarball "https://github.com/nix-community/NUR/archive/master.tar.gz") {};
in
{
  imports = [
    nur-no-pkgs.repos.shamilton.modules.hmModules.redshift-auto
  ];
}
```

### An overlay
In `~/.config/nixpkgs/home.nix` or `/etc/nixos/configuration.nix` :
```nix
let
  # Please edit this and replace with a fixed revision
  # and sha256
  nur-no-pkgs = import (builtins.fetchTarball "https://github.com/nix-community/NUR/archive/master.tar.gz") {};
in
{
  nixpkgs.overlays = [
    nur-no-pkgs.repos.shamilton.overlays.tabbed
    nur-no-pkgs.repos.shamilton.overlays.alacritty
  ];
}
```
